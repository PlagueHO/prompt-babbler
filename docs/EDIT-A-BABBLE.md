## Overview

After you save a babble, Prompt Babbler opens the babble detail page. This page is where you review the saved text, edit it, continue recording, and generate prompts.

![Saved babble detail page](/images/babble-detail.png)

## Header actions

The top row focuses on managing the saved babble:

* **Edit title** changes the current title inline.
* **Generate title from babble text** asks the model to suggest a better title.
* **Continue Babble** returns to the recording page in append mode.
* **Edit** opens the text editor for the existing babble.
* **Delete** removes the babble and its generated prompts.

## Editing the babble

Select **Edit** to modify the saved text and tags.

![Editing a babble](/images/babble-edit.png)

In edit mode you can:

* update the transcript text directly
* add or remove tags
* save the changes
* cancel without applying them

## Generate Prompt section

The prompt-generation area lets you reuse the same babble in different ways:

* choose a template
* choose **Text** or **Markdown** output
* optionally allow emojis
* select **Generate Prompt**

When generation finishes, the new prompt appears immediately on the page.

![Generated prompt and prompt history](/images/babble-prompt-generated.png)

## Prompt history

Every generated prompt is stored under the babble:

* each history entry shows the template name and generation time
* you can reopen the stored prompt text
* **Copy** copies a prompt to the clipboard
* **Regenerate** creates a new version with the same template
* **Delete** removes an old generated prompt

## When to use this page

Use the babble detail page when you want to:

* clean up a transcript after recording
* keep adding to an idea over time
* experiment with different templates from the same source text
* reuse prompt history without recording the idea again
