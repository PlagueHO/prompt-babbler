## Quick workflow

Prompt Babbler turns a short spoken note into a reusable AI prompt in one flow:

1. Open **New Babble**.
1. Record your thoughts.
1. Save the babble.
1. Generate a prompt with a built-in template.
1. Copy the result or keep it in prompt history for later.

![Recorded babble ready to save](/images/create-babble-recorded.png)

## Record your first babble

1. From the top navigation, select **New Babble**.
1. Enter a title if you want one. If you leave it blank, Prompt Babbler creates a default title when you save.
1. Add tags if they will help you find the babble later.
1. Select the microphone button and speak naturally.
1. Stop recording when the transcript looks complete.
1. Select **Save Babble**.

> [!TIP]
> Short, direct recordings usually produce the cleanest prompts. You do not need to speak in polished prose.

## Generate a prompt with a built-in template

After the babble is saved, Prompt Babbler opens the babble detail page.

1. In **Generate Prompt**, keep the default built-in template or choose a different one.
1. Choose **Text** or **Markdown** output.
1. Enable **Allow emojis** only if your target system benefits from them.
1. Select **Generate Prompt**.
1. When the output appears, select **Copy** if you want it on your clipboard.

![Generated prompt on a saved babble](/images/babble-prompt-generated.png)

## What happens next

Prompt Babbler keeps both the original babble and the generated prompt:

* The babble appears on the **Home** page.
* The generated result appears in **Prompt History** on the babble page.
* You can return later to edit the text, continue recording, or generate a different prompt from the same babble.
