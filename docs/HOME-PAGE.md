## Overview

The Home page is the main index for everything you have recorded. It gives you a quick way to create a new babble, reopen saved items, and organize older content.

![Home page with saved babbles](/images/home-with-babbles.png)

## Main areas

### Top navigation

The header is available across the app:

* **Prompt Babbler** returns you to Home.
* **Home** opens the babble index.
* **New Babble** opens the recording screen.
* **Templates** opens prompt-template management.
* **Search babbles** opens the semantic search command palette.
* The user menu shows the current authentication mode. In anonymous mode it appears as **Anonymous**.

### Your Babbles section

The top of the page introduces the app and gives you a primary **New Babble** action. This is the fastest way to start a new recording.

### Featured babble cards

When you have recent or pinned babbles, Prompt Babbler shows them as cards near the top of the page:

* Selecting a card opens the babble detail page.
* The date shows when the babble was last updated.
* The card preview shows a shortened version of the babble text.
* The pin button keeps important babbles easy to reach.

### Older Babbles list

The lower list is designed for browsing and management:

* **Filter babbles** narrows the list by text.
* **Sort** switches between date and title.
* The sort direction can be ascending or descending.
* Each row opens the babble detail page.
* The pin action is available from the list as well.

## Empty state

On a first run, the Home page shows a simplified empty state with a call to record your first babble.

![Home page empty state](/images/home-empty.png)

Use the empty-state button or the top navigation to move directly to the recording screen.

## Search behavior

The Home page includes two ways to find content:

* The list filter searches within the visible babble list.
* The header search opens the semantic search dialog, which is useful when you remember an idea but not the exact title.

## Best uses for the Home page

Use Home when you want to:

* start a new recording quickly
* reopen an existing babble
* pin important babbles
* sort or filter older content
* jump into semantic search
